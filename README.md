# Clinical Research Data Analysis Portfolio - Healthcare Patient Outcomes

## Project Overview
End-to-end data analysis project demonstrating how to take messy clinical healthcare data, clean it, extract valuable insights with SQL, and build an interactive dashboard in Power BI. 

This project mirrors the exact work of a **Clinical Research Data Coordinator** — abstracting clinical data, creating reports, monitoring outcomes, and supporting research/regulatory needs (perfect for Mayo Clinic and similar healthcare data roles in Scottsdale/Phoenix).

**Tools Used**: 
- Excel + Power Query (data cleaning)
- SQL (insights & analysis)
- Power BI (interactive visualizations & storytelling)

## Business Problem
Healthcare data is often messy and inconsistent (duplicates, mixed formats, missing values). This project shows how to transform raw data into actionable insights that can improve patient outcomes, program effectiveness, and clinical decision-making.

## Skills Demonstrated
- **Data Cleaning**: Handled duplicates, inconsistent dates, text cleaning, data type conversions, and missing values using Power Query.
- **SQL Analysis**: Joins, aggregations, CTEs, window functions to uncover trends in patient conditions and treatment effectiveness.
- **Power BI Dashboard**: Interactive visuals, KPIs, slicers, and storytelling for stakeholders.
- **End-to-End Thinking**: From raw messy data to business/clinical recommendations.
